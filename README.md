# HE-Voting

An online voting system still in development using [Drogon](https://github.com/drogonframework/drogon), [Microsoft SEAL](https://github.com/microsoft/SEAL), [Angular](https://angular.io/) and [node-seal](https://github.com/s0l0ist/node-seal). 

This is an almost-functional project that is part of my Bachelor thesis. Further documentation will be added when it's been brought up to a higher standard. Uploaded to GitHub for remote contributing.
